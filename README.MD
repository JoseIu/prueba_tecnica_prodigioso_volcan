## Technologies: React, JS, TS , SASS, Node.js, Express, MongoDB Docker

1.- Configura el archivo `.env` con los datos proporcionados por email: `CLIENT_ID`. `VITE_BACK_URL` y `CLIENT_SECRET`.

4.- Entrar en http://localhost:9778
